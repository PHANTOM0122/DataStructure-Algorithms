# 1) Selection Sort
- **Array에서 가장 min element를 찾은 후에(O(N)), array의 가장 앞으로 swap한다. 이 과정을 numElement(O(N))만큼 반복한다**
- Time complexity : (N-1) + (N-2) + ... + 2 + 1 = O(N*N)
